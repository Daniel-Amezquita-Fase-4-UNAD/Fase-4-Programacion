# Fase_4_Programación
Arquitectura — el diagrama de árbol de clases explica la jerarquía de un vistazo.
Principios OOP — tabla que mapea cada principio a su implementación concreta, útil si es para una entrega académica.
Excepciones personalizadas — tabla con código, clase y cuándo se lanza cada una.
Ciclo de vida de la reserva — el diagrama de estados aclara qué transiciones son válidas.

Ejemplos de uso — código listo para copiar y probar.
